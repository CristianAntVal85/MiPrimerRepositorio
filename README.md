# MiPrimerRepositorio
# Portfolio Web – Cristian Antón (Freelancer)

Este proyecto es una página web personal desarrollada con **HTML** y **CSS**, 
creada para presentarme como desarrollador web freelance. 
El sitio incluye una sección de servicios, información personal, ubicación y 
un formulario de contacto.

## 🚀 Tecnologías utilizadas
- **HTML5** – Estructura del contenido.
- **CSS3** – Estilos, diseño visual y maquetación.
- **Normalize.css** – Para asegurar consistencia entre navegadores.
- **Google Fonts** – Tipografías personalizadas.

## 📌 Características principales
- Diseño limpio y moderno.
- Sección de servicios (Diseño web, Aplicaciones web, E‑Commerce).
- Hero con ubicación y llamada a la acción.
- Navegación superior responsiva.
- Formulario de contacto estructurado.
- Footer con derechos de autor.

## 📁 Estructura del proyecto
├── index.html
└── css/
├── normalize.css
└── style.css


## 🌍 Objetivo del proyecto
Este sitio forma parte de mi portfolio como desarrollador freelance y sirve como carta 
de presentación para futuros clientes, mostrando mis habilidades y estilo de trabajo.
